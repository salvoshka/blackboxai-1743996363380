# Vilea's Project - Student Task Assistant

## Project Overview
Vilea's Project is a web-based application designed to empower students by helping them manage academic tasks with a focus on collaboration and support. The platform offers various features that make it easy for users to stay organized, including task management, smart scheduling, and collaborative tools.

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/vileas-project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vileas-project
   ```

3. Open `index.html` in your preferred web browser.

## Usage
Once you have installed the project, open `index.html` in a web browser to view the application. Users can interact with various sections including features, task upload, and contact forms.

## Features
- **Task Management**: Keep track of assignments and deadlines seamlessly.
- **Smart Scheduling**: Automatically schedule your study sessions based on your subject workload.
- **Collaboration**: Easy connection with classmates for forming study groups.

## Dependencies
The project utilizes the following dependencies:
- **Tailwind CSS**: For styling and layout.
- **Font Awesome**: For icons.
- **Google Fonts**: Specifically, the Poppins font.

No additional JavaScript packages or libraries outside of these are included in the project.

## Project Structure
The project directory contains the following structure:

```
vileas-project/
├── index.html         # Main HTML file
└── index_backup.html  # Backup HTML file (not in use)
```

### HTML Structure Overview
- **Header**: Contains navigation links and branding.
- **Hero Section**: Introduces the core purpose of the project.
- **Features Section**: Highlights the tools and features available to users.
- **Upload Task Section**: Form for users to upload their academic tasks.
- **Contact Section**: Ways for users to reach out for support.
- **Footer**: Contains additional links and copyright information.
